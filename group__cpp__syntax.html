struct tinker::PtrTrait

template<class PTR>
struct tinker::PtrTrait< PTR >

A helper class to get the traits of the given 1D or 2D pointer type.

Example:

PTR PtrTrait<PTR>::type PtrTrait<PTR>::n
float* float 1
int (*)[3] int 3
